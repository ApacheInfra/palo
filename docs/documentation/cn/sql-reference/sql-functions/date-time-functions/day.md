# day

## Syntax

`INT DAY(DATETIME date)`

## Description

获得日期中的天信息，返回值范围从1-31。

参数为Date或者Datetime类型

## Examples

```
mysql> select year('1987-01-31');
+-----------------------------+
| year('1987-01-31 00:00:00') |
+-----------------------------+
|                           31|
+-----------------------------+
```
