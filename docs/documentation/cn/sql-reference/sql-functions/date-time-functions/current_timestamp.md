# current_timestamp
## description

获得当前的时间，以Datetime类型返回

 Syntax

`DATETIME CURRENT_TIMESTAMP()`

## example

```
mysql> select current_timestamp();
+---------------------+
| current_timestamp() |
+---------------------+
| 2019-05-27 15:59:33 |
+---------------------+
