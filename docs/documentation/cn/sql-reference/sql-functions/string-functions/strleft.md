# strleft
## description

它返回具有指定长度的字符串的左边部分

 Syntax

`VARCHAR strleft(VARCHAR str)`

## example

```
mysql> select strleft("Hello doris",5);
+------------------------+
| strleft('Hello doris', 5) |
+------------------------+
| Hello                  |
+------------------------+
```
##keyword
STRLEFT,STRLEFT
